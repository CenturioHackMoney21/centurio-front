# Centurio - HackMoney2021

### Prepare project

Run `yarn install`.

Run `yarn prepare`.

### Development

Run `yarn dev`.

### Production

Run `yarn build`. The generated files will be on the `dist` folder.

### Testing

Run `yarn test`. Tests are performed on production build, so be sure to build your app first.
